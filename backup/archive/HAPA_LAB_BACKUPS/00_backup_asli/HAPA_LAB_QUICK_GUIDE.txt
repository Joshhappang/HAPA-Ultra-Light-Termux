===============================
HAPA_LAB – Panduan Cepat (HP SE)
===============================

1. Struktur Folder HAPA_LAB
~/HAPA_LAB/
├── key1/
├── key2/
├── key3/
├── multisig_msg*.txt
├── testnet_tx/
├── testnet_tx_advanced/
├── testnet_tx_multi/
├── auto_return_multi.sh
├── restore_and_run.sh
├── HAPA_MULTISIG_DIAGRAM.txt
├── HAPA_TESTNET_DIAGRAM.txt
├── HAPA_TESTNET_3of3_DIAGRAM.txt
├── HAPA_TESTNET_ADVANCED_DIAGRAM.txt
└── HAPA_LAB_FINAL_SUPER_BACKUP.tar.gz

2. Backup Lengkap HAPA_LAB
cd ~/HAPA_LAB
tar -czf HAPA_LAB_FINAL_SUPER_BACKUP.tar.gz \
key1 key2 key3 \
multisig_msg*.txt \
HAPA_MULTISIG_DIAGRAM.txt \
HAPA_TESTNET_DIAGRAM.txt \
HAPA_TESTNET_3of3_DIAGRAM.txt \
HAPA_TESTNET_ADVANCED_DIAGRAM.txt \
testnet_tx \
testnet_tx_advanced \
testnet_tx_multi \
auto_return_multi.sh

3. Restore & Jalankan Simulasi Otomatis
cd ~/HAPA_LAB
./restore_and_run.sh

4. Jalankan Simulasi Manual
cd ~/HAPA_LAB
chmod +x auto_return_multi.sh
./auto_return_multi.sh

5. Tips Membaca Diagram di HP SE
less HAPA_MULTISIG_DIAGRAM.txt
less HAPA_TESTNET_MULTI_TX_DIAGRAM.txt

6. Catatan Penting
- Semua transaksi lokal, Genesis aman
- Auto-return berjalan jika Key ketiga tidak menandatangani
- Multi-Sig: minimal 2 dari 3 Key menandatangani untuk validasi
