HAPA SYSTEM — INTERNAL vs EXTERNAL (FINAL)

USER → Dashboard / Web (External)
- Read / Request saja
- Menampilkan data & menyampaikan permintaan
- Tidak berkuasa, tidak memutuskan
- Boleh berubah, tapi patuh aturan

Node (Internal)
- Menilai transaksi
- Menjalankan aturan

Ledger (Internal)
- Mengikat kebenaran final
- Tidak bisa diganti

Tombol / UI Control → OFF selamanya
- Tidak ada approve / reject / edit / restart / admin

Prinsip Kunci:
Web = jendela & kurir, Node = hakim, Ledger = memori
